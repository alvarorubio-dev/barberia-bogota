# 💈 Barbería Bogotá

Bienvenido al repositorio del proyecto web para **Barbería Bogotá**. Este proyecto es una aplicación web moderna, rápida y responsiva diseñada para mostrar los servicios, agendar citas y presentar el portafolio de la barbería.

## 🚀 Tecnologías utilizadas

Este proyecto fue construido con las siguientes herramientas:

- ⚛️ **React:** Librería para construir la interfaz de usuario.
- 📘 **TypeScript:** Para un código más seguro y escalable.
- ⚡ **Vite:** Herramienta de construcción ultrarrápida para el entorno de desarrollo.
- 🎨 **CSS / HTML5:** Para el diseño y estructura responsiva.
